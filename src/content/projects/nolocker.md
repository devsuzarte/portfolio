longDescription: "Nolocker resolve um problema cotidiano: PDFs protegidos por senha que precisam ser compartilhados com outras pessoas. O processamento acontece inteiramente no browser via Web APIs, garantindo que o documento nunca sai do dispositivo do usuário. A API em Fastify serve apenas como gateway de autenticação e rate limiting. Deploy automatizado via GitHub Actions para a Vercel com preview por PR."
