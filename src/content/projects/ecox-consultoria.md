longDescription: "Projeto fullstack end-to-end para a EcoX Consultoria Ambiental. Substituí um site WordPress estático por uma stack moderna: Next.js no frontend com Tailwind CSS e Shadcn/UI para design system consistente, PayloadCMS como CMS headless permitindo gestão autônoma de conteúdo pelo cliente, e Fastify como camada de API. Implementei automações de negócio via n8n conectadas à Claude API para geração assistida de relatórios ambientais. O deploy acontece via GitHub Actions para um VPS Ubuntu gerenciado pelo Coolify, com domínio e CDN no Cloudflare para performance global."
