## 🎬 7. CONVERTER VÍDEO EM FRAMES (Para Scroll Animation)

### 🔧 Ferramentas Online Gratuitas:

#### **EZGIF** ⭐
- **URL:** https://ezgif.com/video-to-jpg
- **Features:**
  - Upload vídeo MP4
  - Escolher FPS (recomendo 15-20 para web)
  - Download ZIP com todos os frames
- **Limite:** 100MB

#### **Online-Convert**
- **URL:** https://video.online-convert.com/convert-video-to-images
- **Features:** Mais opções de formato

### 🔧 Ferramentas Locais (Grátis):

#### **FFmpeg** (Command Line) ⭐
```bash
# Instalar FFmpeg
# Windows: https://ffmpeg.org/download.html
# Mac: brew install ffmpeg
# Linux: sudo apt install ffmpeg

# Extrair frames (15 FPS)
ffmpeg -i input.mp4 -vf fps=15 output_%04d.jpg

# Extrair frames otimizados (redimensionar para 1920px width)
ffmpeg -i input.mp4 -vf "fps=15,scale=1920:-1" frame_%04d.jpg

# Converter para WebP (menor tamanho)
ffmpeg -i input.mp4 -vf fps=15 -c:v libwebp frame_%04d.webp
```
