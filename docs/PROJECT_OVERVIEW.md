# Project Overview - Portfolio Juan Benjamin

## 🎯 Objetivo
Portfolio de alto padrão para Juan Benjamin Almeida Suzarte, desenvolvedor full stack, com animações refinadas e identidade visual limpa e elegante — focado em demonstrar capacidade técnica e senso estético apurado.

## 🛠️ Stack Tecnológica
- **Framework:** Astro 4.x (SSG)
- **Page Transitions:** Barba.js
- **Animations:** GSAP + ScrollTrigger
- **3D:** Three.js
- **Styling:** Tailwind CSS + SCSS
- **Language:** TypeScript (strict)

## 🎨 Estilo Visual
**Coeso e editorial** — paleta única de tons de café com variação entre dark/light por seção:
- **Paleta principal:** Espresso, Caramel, Cream, Fog
- **Tipografia:** Cormorant Garamond (display serif) + Inter (body) + JetBrains Mono (code)
- **Efeitos:** grain texture, underline reveal, clip-path image reveals, shadow warm-tinted
- **Animações:** Line reveal, stagger fade, parallax suave, cursor follower, magnetic buttons
